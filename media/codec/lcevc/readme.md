# LCEVC 低复杂性增强视频编码

## 简介

低复杂性增强视频编码（Low Complexity Enhancement Video Coding - LCEVC）是由MPEG以项目名称MPEG-5 Part 2 LCEVC开发的ISO / IEC视频编码标准。
MPEG-5 Part 2 LCEVC（ISO/IEC 23094-2）已由ISO和IEC正式发布。
